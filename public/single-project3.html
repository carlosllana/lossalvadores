Tarquinia NFT

Desarrollo de plataforma de NFT.

En este proyecto desarrollamos la primera galería NFT chilena. Para sumar valor a cada token NFT agregamos funcionalidades de realidad aumentada y realidad virtual a cada cuadro. Utilizamos la Blockchain de Ethereum para crear los Tokens
Dividimos el proyecto en etapas de integración y desarrollo con el objetivo de comenzar a vender antes de desarrollar.
